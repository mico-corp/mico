## Dependencies

### DVS Abstaction Layer

Installation from source code: 

```
git clone https://github.com/mgrova/dvsal.git
cd dvsal
mkdir build ; cd build
cmake .. -DCMAKE_BUILD_TYPE=Release ; make -j
sudo make install
```
