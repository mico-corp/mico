Math Plugin tutorials {#tutorial_math_mplugin}
=====================

The math plugin has following implementations:

* Function providers
    1. Timer
    2. Sine
    3. Cosine
    4. Const

* Operation blocks
    1. Comparison
    2. Sum
    3. Substract
    4. Multiply
    5. Division
    6. Square root
    7. Pow
